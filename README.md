# Libasm

42 Project | Small assembly library — reimplements common C library functions in x86_64 assembly.

## Build
```bash
make
```

## Files
- Core: `ft_read.s`, `ft_write.s`, `ft_strlen.s`, `ft_strcpy.s`, `ft_strdup.s`, `ft_strcmp.s`
- Bonus: `ft_atoi_base_bonus.s`, `ft_list_*.s`

## Notes
Intended for learning assembly, not production use.
